# Project - Portfolio Data Clearinghouse

A simplified portfolio data reconciliation system.

## Quick start

`git clone https://github.com/dhindanov/portfolio-clearinghouse.git`

`cd portfolio-clearinghouse`

`docker compose up --build -d`

Navigate to the swagger documentation:
`http://localhost:5000/openapi/swagger`

Or to the sample UI:
`http://localhost:5000/`
